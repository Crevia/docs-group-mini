## Common Questions

**How do I join a group?**
Browse available groups from the home page. If a group is open, you can request to join. The group admin will review and approve your request.

**How do I set an active account?**
Go to the account details page. Use the "Current Account" switch to set it as your active account. The account must have an active status before it can be set as current.

**Who can invite members to a group?**
Group admins and moderators can send invitations. The group must have an amount and start date configured before invitations can be sent.
